## Proposal #48: Ground Truth Protocol Category Tagging

**Source:** Entry 56 (procedural vs behavioral gap)
**Priority:** Medium
**Scope:** DSM_0.2 Appendix C.1 (Experiment Protocol)

Add a `category` field to ground truth section lists in experiment definitions:
`procedural` (observable named actions) or `behavioral` (implicit quality
standards). This allows experiments to report separate recall scores per
category and prevents behavioral protocols from masking procedural extraction
quality.

## Proposal #49: Stable Identifiers for Ground Truth References

**Source:** Entry 57 (ID stability)
**Priority:** Low
**Scope:** DSM_0.2 Appendix C.1 (Experiment Protocol)

Recommend that experiment ground truth identifiers use protocol titles
(e.g., "Session Transcript Protocol") rather than section-number-derived slugs
(e.g., "7-session-transcript-protocol"). This prevents breakage when documents
are renumbered during evolution.
