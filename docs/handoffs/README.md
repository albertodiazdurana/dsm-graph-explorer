# docs/handoffs/

Session handoff documents for continuity between work sessions.

## Files

| File | Purpose |
|------|---------|
| `handoff_graph_explorer.md` | Initial project setup handoff |

## Usage

Created at end of session if work is incomplete. Contains:
- What was done
- What's next
- Open questions
- How to resume

**Note:** For longer breaks, use checkpoints/ instead. Handoffs are for short session boundaries.
